# Task Manager API

A scalable REST API with authentication, role-based access, and CRUD operations.

## Features
- JWT Authentication
- Role-based access control
- Task CRUD system
- Swagger documentation
- React frontend

## Tech Stack
- Node.js, Express
- MongoDB, Mongoose
- React.js
- JWT, bcrypt

## Setup

### Backend
npm install
npm run dev

### Frontend
npm install
npm start
